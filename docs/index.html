Getting Started with TeX-Rex

TeX-Rex is an AI-powered academic paper analysis tool that helps researchers improve their scientific manuscripts through automated feedback and suggestions.

Prerequisites

  • Python 3.8 or higher
  • OpenAI API key
  • Basic understanding of academic paper structure

Installation

pip install tex-rex

Configuration


# Set your OpenAI API key
import os
os.environ["OPENAI_API_KEY"] = "your-api-key-here"

Components

Abstract Analyzer

The Abstract Analyzer evaluates the clarity, completeness, and impact of your paper's abstract.

Features

  • Checks for all essential components (background, methods, results, conclusions)
  • Evaluates clarity and conciseness
  • Suggests improvements for impact

Usage


from abstract.abstract_maker import AbstractAnalyzer

analyzer = AbstractAnalyzer(api_key="your-api-key")
result = await analyzer.analyze_abstract(text)
analyzer.display_results(result)

Introduction Analyzer

The Introduction Analyzer assesses the background information, research gap identification, and hypothesis presentation.

Features

Usage

API Reference

Each analyzer component follows a similar API pattern:

Common Methods

  • analyze_*(text): Main analysis method
  • display_results(result): Display analysis results
  • get_suggestions(): Get improvement suggestions

Examples

Complete Paper Analysis


# Example of analyzing a complete paper
from tex_rex import TexRex

analyzer = TexRex(api_key="your-api-key")

# Analyze abstract
abstract_result = await analyzer.analyze_abstract(abstract_text)

# Analyze introduction
intro_result = await analyzer.analyze_introduction(intro_text)

# Analyze methods
methods_result = await analyzer.analyze_methods(methods_text)

# Display combined results
analyzer.display_all_results()
